# 🌾 Agro Paraná - Portal Inteligente

O **Agro Paraná** é um portal web informativo sobre o agronegócio paranaense, apresentando dados históricos, econômicos e produtivos do estado do Paraná de forma moderna, interativa e responsiva.

## 📖 Sobre o Projeto

O projeto foi desenvolvido para apresentar informações relevantes sobre o setor agrícola do Paraná, um dos maiores polos do agronegócio brasileiro.

A plataforma reúne:

- História da agricultura paranaense
- Dados econômicos do agronegócio
- Mapa interativo
- Dashboard com gráficos de produção
- Curiosidades sobre o setor
- Galeria de imagens do campo
- Interface responsiva
- Modo escuro

---

## 🚀 Funcionalidades

### 🌱 Informações Institucionais
Apresenta uma visão geral do agronegócio do Paraná, destacando sua importância para a economia nacional.

### 📜 História do Agro
Linha histórica mostrando a evolução da agricultura desde os povos indígenas até a agricultura de precisão atual.

### 💰 Economia Agrícola
Exibe informações sobre:

- Produção de soja e milho
- Liderança na produção de frango
- Exportações agrícolas
- Mecanização do campo

### 🗺️ Mapa Interativo
Utiliza a biblioteca **Leaflet.js** para exibir informações geográficas relacionadas ao agronegócio.

### 📊 Dashboard de Produção
Gráficos desenvolvidos com **Chart.js** para visualização de dados agrícolas.

### 🧠 Curiosidades
Seção com informações rápidas sobre tecnologia, produtividade e inovação no campo.

### 🌦️ Clima e Produção
Explica a influência do clima subtropical do Paraná na produção agrícola.

### 📸 Galeria de Imagens
Exibe imagens representativas do setor agrícola.

### 🌙 Modo Escuro
Permite alternar entre tema claro e escuro para melhor experiência do usuário.
